Highlight Options — Distribución de Ventas

3 propuestas para resaltar información clave en el gráfico de porcentaje relativo por sucursal

Opción A

Top 3 & Bottom 3

Las 3 sucursales con mayor contribución se destacan en azul primario con badge de ranking. Las 3 con menor contribución se marcan en terracota (color terciario del design system) para señalar oportunidad de mejora. Las intermedias quedan en azul tenue.

Opción B

Gradient Intensity

Todas las barras usan el mismo color primario pero con opacidad proporcional a su ranking. La sucursal #1 tiene opacidad 100%, y cada posición inferior pierde intensidad progresivamente. Se agrega una línea divisoria «Promedio» para marcar el punto de corte natural. Sin categorías fijas — el diseño escala a cualquier número de sucursales.

Opción C

Above / Below Average

El promedio se calcula automáticamente (100% ÷ n sucursales). Toda sucursal por encima del promedio se muestra con barra sólida + texto azul; las que están debajo se atenúan. Una línea punteada marca el umbral y al pie se muestran 2 chips resumen: cuántas están arriba y cuántas abajo. Adaptativo — no depende de un conteo fijo.
